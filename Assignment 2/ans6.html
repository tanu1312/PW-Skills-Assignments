1. HTML (Hypertext Markup Language)

    Primary Use Cases Key Features/Benefits
    Structuring web content Defines the structure of web pages using elements like headings, paragraphs, lists, and links.
    Creating forms Allows users to input data through various form elements like text fields, checkboxes, and buttons.

  2. CSS (Cascading Style Sheets)

    Primary Use Cases Key Features/Benefits
    Styling web content Applies styles (e.g., colors, fonts, layouts) to HTML elements to control the visual presentation of a webpage.
    Creating responsive designs Enables the design to adapt to different screen sizes and devices using media queries and flexbox/grid layouts.

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    Adding interactivity Enhances web pages with interactive features such as animations, forms validation, and dynamic content loading.
    Handling user input Captures and processes user actions, such as clicks and keyboard input, to create responsive web applications.
