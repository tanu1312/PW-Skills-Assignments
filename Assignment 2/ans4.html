Orderded list of HTML Tags

  1. html
      2. The root element that wraps all content on a web page.
      Contains metadata about the document and defines the document type.
  2. body
      3. Description: Contains the visible content of the web page, including text, images, and other elements.
  3. head
      4. Description: Contains meta-information about the document, such as the title, character encoding, and linked stylesheets.
  4. title
      5. Description: Sets the title of the web page, displayed in the browser's title bar or tab.
  5. script
      6. Description: Embeds or references JavaScript code in the HTML document.
